Documentação de manutenção do website do Xubuntu

Lançamentos

O site do Xubuntu rastreia lançamentos com uma taxonomia personalizada do WordPress. Os termos de lançamento podem ser anexados virtualmente para qualquer tipo de conteúdo usado no site. Informações do termo de lançamento e conteúdo anexado são usados para automatizar a criação de algum conteúdo no site, incluindo, mas não necessariamente limitado, a: páginas de lançamento , página de Download , página de Ajuda e Suporte e a página Na Imprensa .

Para editar a informação genérica de cada lançamento (nome de código, data de lançamento, data do EOL, links para downloads de torrents, documentação online e alternância de lançamento), vá para Postagens – Lançamentos .

Para anexar qualquer conteúdo a uma versão, selecione as versões desejadas na página de edição de conteúdo.

Os seguintes códigos de acesso são usados para gerar informações agregadas sobre lançamentos (substituir xx-xxpelos dados do lançamento desejado):

  • [torrents release=xx-xx]para produzir links de download Torrent

  • [mirrors release=xx-xx] para produzir lista de links de download de mirrors com informação estendida disponível

  • [press release=xx-xx]para produzir todos os links de imprensa anexados ao lançamento e [pressione] para produzir o arquivo de imprensa

  • [documentation_links] para produzir uma lista com links para a documentação online para todas as versões lançadas e suportadas

A página de lançamento das versões usa uma página modelo com o mesmo nome; o modelo lista todas as versões lançadas e suportadas com uma pequena descrição. As páginas de lançamento são criadas automaticamente.

Links de lançamento

Para adicionar links de imprensa ( e outros links oficiais de versões específicas além dos links de download Torrent e links para a documentação online), vá para links de lançamento. Uma vez adicionados ao banco de dados,links irão aparecer nos lugares apropriados automaticamente.

Espelhos de download

As listas de links de download são criadas automaticamente a partir dos modelos de URL salvos no banco de dados. Para gerenciar os mirrors de lançamento nas listas de download,vá para Mirrors DL. Na edição de páginas você é capaz de tornar mirrors de download ativos ou inativos (respectivamente visíveis e ocultos no front-end). Poderá também alterar quais links de download são exibidos na listagem padrão,configure o mirror como um mirror de download principal se quiser que ele seja visível diretamente.

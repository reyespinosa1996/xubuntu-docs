Introdução à apresentação do Xubuntu na web

Xubuntu.org

O site é a principal fonte de informação para usuários finais. O conteúdo do site deve ser facilmente compreensível e acessível.

Este site é uma instância do WordPress em um servidor mantido pela Canonical IS. O conteúdo é supervisionado e administrado pelo time do website do Xubuntu. Este time pode abrir tickets para a Canonical IS, em caso de problemas. Bugs que são relatados no site podem ser reportados ao Launchpad, no endereço bugs for xubuntu-website.

O código fonte do website está disponível em lp:xubuntu-website(o tema) e lp:xubuntu-website/plugins(os plugins).

Xubuntu Wiki

A Xubuntu wiki é usada para algumas comunicações entre desenvolvedores e serve como um repositório de antigos quadros de referência de versão e especificações.

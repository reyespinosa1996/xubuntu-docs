Conjunto de pacotes do Xubuntu

A equipe do Xubuntu mantêm um subconjunto de pacotes no arquivo do Ubuntu. O conjunto de pacotes do Xubuntu para a versão em desenvolvimento atual pode ser achado listado no arquivo.

PPAs de desenvolvimento

The Xubuntu developers use several PPAs (personal package archives) for staging, testing and previewing new packages and package versions. These PPAs can be found under the Xubuntu Developers page on Launchpad.

No momento,os PPAs oficiais usados para desenvolvimento são:

PPA Descrição
Builds diárias do Xubuntu Builds diárias Git/Bzr para pacotes relacionados ao Xubuntu e/ou Xfce4.
Controle de qualidade experimental do Xubuntu Um PPA experimental para o Xubuntu. Os pacotes e versões de pacotes neste PPA estão sendo preparados para inclusão no Xubuntu. Os pacotes são enviados para a conveniência das pessoas testando essas novas versões de pacotes e funções. Alguns deles são builds diárias usadas em testes.
Preparação do controle de qualidade do Xubuntu Um PPA de preparação para o Xubuntu, normalmente usado para a preparação SRU. Os pacotes e versões de pacotes neste PPA estão sendo preparados para a inclusão no Xubuntu. Os pacotes são enviados para a conveniência das pessoas testando essas novas versões de pacotes e funções. Alguns deles são builds diárias usadas em testes.
Xubuntu Extras Um PPA de visualização para o Xubuntu. Os pacotes neste PPA são considerados para a inclusão nos repositórios do Ubuntu e/ou Xubuntu posteriormente. Os pacotes são enviados para a conveniência das pessoas dispostas a visualizar as novas funções.

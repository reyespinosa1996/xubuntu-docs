Obtendo e atualizando branches do Bzr

Depois que você configurar Bzr, você pode obter uma cópia local de uma branch bzr branch LOCALIZAÇÃO. Isto irá criar um diretório para a branch.

Para ter certeza que sua cópia local está atualizada, execute bzr pull dentro do diretório da branch.

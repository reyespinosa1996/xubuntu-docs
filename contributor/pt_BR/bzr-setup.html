Configurando o Bzr

Para configurar Bzr, faça o seguinte:

  • Para ter certeza que Bzr está instalado em seu sistema, execute sudo apt-get install bzr

  • Para configurar suas informações pessoais para usar no Bzr, execute bzr whoami "John Doe <john.doe@gmail.com>"

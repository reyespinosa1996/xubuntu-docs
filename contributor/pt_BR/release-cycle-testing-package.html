Testes de pacotes

Quando sendo usado a equipe de controle de qualidade irá agendar um sprint de testes de pacotes durante o ciclo entre testes de marcos para garantir que os aplicativos que são incluídos no Xubuntu tenham passado por testes o suficiente.

Independentemente de um teste de pacote estar sendo realizado durante um ciclo de desenvolvimento, testes adicionais devem ser conduzidos para um novo aplicativo padrão ser incluído na próxima versão do Xubuntu, quando um aplicativo existente tem uma atualização consideravelmente grande no meio do ciclo ou quando os desenvolvedores solicitarem testes específicos para uma atualização específica de um aplicativo.

Chamadas para sprints de testes de pacotes serão enviadas como requerido.

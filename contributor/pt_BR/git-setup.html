Configurando o Git

Para configurar o Git,faça o seguinte:

  • Para ter certeza que Git está instalado em seu sistema, execute:

    sudo apt-get install git

  • Para configurar as suas informações pessoais para usar no Git,execute:

    git config --global user.name "John Doe"

    git config --global user.email "john.doe@gmail.com"

Infraestrutura de teste

Preparando-se para testar

Para testar o Xubuntu com sucesso,você irá precisar no mínimo de 4 coisas: uma conta Launchpad, inscrição na lista de e-mails,uma imagem e algo para testá-la. Esta página irá dar a você as informações básicas para fazer essas coisas e mais.

Contatos

Você precisa de uma conta Launchpad,para se unir a(s) equipes necessárias nele,e logar nos 2 rastreadores do controle de qualidade.

Nós usamos uma lista de e-mails para detalhar o que pretendemos testar durante um ciclo, usamos a mesma lista na chamada para testes. Essa lista também é usada pela equipe para discutir o desenvolvimento do Xubuntu. Você precisará se unir a lista PostagemXubuntu Devel.

Embora não seja necessário fazê-lo, fazemos muito uso do IRC, então tendo uma conta você poderá usá-la para cessar o principal canal #xubuntu-devel para lhe ajudar, alternativamente você pode acessar o canal através da aba IRC do rastreador do Xubuntu desenvolvimento.

Se junte a equipe de Testadores do Xubuntu Launchpad.

Considere adicionar os seus detalhes a página Equipamento. Podemos usar essa informação para solicitar testes específico de testadores específicos.

Obtendo a ISO

ISOs podem ser obtidas através de alguns métodos. O método mais simples é visitar o rastreador ISO. Siga os casos de testes através do codinome de lançamento para o grupo de testes do Xubuntu. Então escolha entre 64 ou 32 bits. Links de download estão disponíveis lá.

Ao testar mais regularmente, você pode achar o uso do link zsync mais apropriado(especialmente quando a largura de banda é um problema). Ele irá apenas puxar a diferença entre a sua cópia local e a atual.

Métodos de teste

Agora que configurou o(s) seus contatos básicos e tem a sua ISO, você está numa posição de testá-la.

Existem dois métodos principais para testar a ISO:

  • Usar uma máquina virtual lhe possibilita a continuar usando uma versão normal quando estiver testando as nossas ISOs. Você pode instalar uma máquina virtual usando diretamente a sua ISO baixada.

  • Instale a ISO em hardware onde essa possibilidade está disponível. Isso é mais importante no final do ciclo de desenvolvimento. Você irá precisar de ter a ISO em alguma mídia inicializável. Um membro da equipe do Website do Xubuntu detalha um método simples para preparar um drive bootável USB com GNOME Discos.

Embora seja possível usar VirtualBox, experiências anteriores demonstraram que problemas encontradas durante os testes no VirtualBox são difíceis de replicar em outro lugar. Você pode querer usar KVM. Membros da equipe de controle de qualidade do Xubuntu possuem experiência com o uso disso para testes e podem ajudar.

Reportando seu resultado

Existem dois rastreadores de QA que usamos.

  • O rastreador de ISO mencionado acima para qualquer teste feito com uma ISO através de uma instalação ou teste de atualização.

  • O rastreador Pacote,que é usado quando estamos testando aplicativos durante um ciclo de desenvolvimento.

Esses rastreadores formam o suporte principal do que nós somos capazes de ganhar dos seus testes. É importante que você relate os seus resultados.

Bem-vindo!

Esta documentação é uma referência a todos colaboradores do Xubuntu. Os capítulos desta documentação fornecem informação sobre os processos de desenvolvimento - tanto sociais quanto técnicos - que os colaboradores do Xubuntu usam como uma diretriz em seu trabalho.

Existem dois apêndices principais para esta documentação:

  • Apêndice A, Documento Estratégico,que é a diretriz primária no desenvolvimento do Xubuntu.

  • Apêndice B, Referência Comum, que descreve várias tarefas técnicas que os desenvolvedores do Xubuntu precisam usar continuamente.

Esta documentação é escrita com a suposição que o usuário conhece as noções básicas de trabalhar com o Xubuntu. Por exemplo,espera-se que o leitor saiba como trabalhar usando um terminal e entenda quando precisar de usar um,sem uma menção explícita.

Obtendo e atualizando repositórios do Git

Após configurar o Git,você pode obter uma cópia local de um repositório ao executar git clone LOCALIZAÇÃO. Isso irá criar um diretório para o repositório.

Para ter certeza de que a sua cópia local de uma branch está atualizada, executegit pulldentro do diretório da branch.

Introducción a la presencia web de Xubuntu

Xubuntu.org

El sitio web está dirigido al usuario final. El contenido del sitio web debe ser accesible y fácil de entender.

El sitio web es una instancia de WordPress en un servidor mantenido por Canonical IS. El sitio web lo supervisa y lo administra el equipo del sitio web de Xubuntu. El grupo del sitio web puede crear tickets dirigidos a Canonical IS en caso de problemas. Los errores relacionados con el sitio web pueden reportarse en Launchpad, en errores de xubuntu-website.

The source code for the website is available at lp:xubuntu-website (the theme) and lp:xubuntu-website/plugins (the plugins).

wiki de Xubuntu

The Xubuntu wiki is used for some developer communication and serves as an archive for older release roadmaps and specification.

Set Paket Xubuntu

Tim Xubuntu mengelola subset paket dalam arsip Ubuntu. Paket Xubuntu untuk rilis pengembangan saat ini dapat ditemukan terdaftar di paket set arsip.

Pengembangan PPA

Pengembang Xubuntu menggunakan beberapa PPA (arsip paket pribadi) untuk menentukan stadium, menguji dan melihat dulu paket dan versi paket baru. PPA ini dapat ditemukan di halaman halaman Xubuntu Developers di Launchpad.

Saat ini, AKP resmi untuk penggunaan pembangunan adalah:

PPA Deskripsi
Xubuntu Daily Builds Git/Bzr setiap hari dibuat untuk paket yang terkait dengan Xubuntu dan/atau Xfce4.
Xubuntu QA Experimental An experimental PPA for Xubuntu. The packages and package versions in this PPA are being prepared for inclusion in Xubuntu. The packages are uploaded for the convenience of people testing these new package versions and features. Some of them are daily builds used in testing.
Xubuntu QA Staging A staging PPA for Xubuntu, often used for SRU preparation. The packages and package versions in this PPA are being prepared for inclusion in Xubuntu. The packages are uploaded for the convenience of people testing these new package versions and features. Some of them are daily builds used in testing.
Xubuntu Extras PPA pratinjau untuk Xubuntu. Paket-paket dalam PPA ini dipertimbangkan untuk dimasukkan dalam repositori Ubuntu dan/atau Xubuntu di lain waktu. Paket-paket tersebut diunggah untuk kenyamanan orang-orang yang ingin melihat fitur baru.

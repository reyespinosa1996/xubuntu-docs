Dokumentasi kontributor

Tim dokumentasi memimpin upaya untuk menulis dokumentasi kontributor. Sementara berbagai subteam bertanggung jawab atas bagian mereka sendiri, tim dokumentasi harus membantu tim untuk mendorong konten mereka ke cabang utama dan jika diperlukan, membantu kontributor untuk memformat konten dalam markup DocBook.

The online contributor documentation should be kept updated especially when major changes are made; the documentation team should request an update whenever they see fit.

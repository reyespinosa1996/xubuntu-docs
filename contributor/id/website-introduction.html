Pengantar kedatangan web Xubuntu

Xubuntu.org

Situs web adalah output utama bagi pengguna akhir. Konten di situs web harus mudah dimengerti dan didekati.

Situs web ini adalah turunan WordPress di server yang dikelola oleh Canonical IS. Konten situs web diawasi dan dikelola oleh tim situs web Xubuntu. Tim situs web dapat mengirimkan tiket ke IS Canonical untuk masalah apa pun. Bug yang terkait dengan situs web dapat dikirimkan di Launchpad, di bawah bug untuk xubuntu-situs web .

The source code for the website is available at lp:xubuntu-website (the theme) and lp:xubuntu-website/plugins (the plugins).

Wiki Xubuntu

The Xubuntu wiki is used for some developer communication and serves as an archive for older release roadmaps and specification.

Tempat-tempat Penting

#xubuntu-devel

Saluran IRC #ubuntu-devel adalah tempat dimana sebagian besar komunikasi dan kerja sama harian tentang pengembangan Xubuntu terjadi. Ini juga tempat untuk mengenal orang dan membuat diri Anda dikenal.

Pelacak Status

Pengembangan untuk rilis Xubuntu saat ini dilacak di Pelacak pengembangan Xubuntu. Selain menunjukkan status saat ini untuk cetak biru dan item pekerjaan masing-masing, pelacak menampilkan beberapa kemajuan umum. Akhirnya, Anda dapat mengakses kalender tim, wiki Xubuntu, milis pengembangan dan bergabung dengan saluran IRC pengembangan # xubuntu-devel darinya.

Menggunakan Git

Git adalah sistem kontrol versi yang digunakan dalam pengembangan Xfce.

Satu perbedaan utama antara Git dan Bazaar untuk diperhatikan adalah bahwa, di Bazaar Anda biasanya hanya berurusan dengan cabang, sedangkan di Git Anda harus membedakan antara cabang dan repositori, yang dapat menampung banyak cabang. Ini juga tercermin dari berbagai perintah yang Anda gunakan. Lain adalah bahwa Git tidak secara otomatis membuat perubahan untuk komit, Anda harus mengatakannya untuk melakukannya.

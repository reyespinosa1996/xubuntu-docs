Mengatur Git

Untuk mengatur Git, lakukan hal berikut:

  • Untuk memastikan Git telah terinstal pada sistem anda, jalankan:

    sudo apt-get install git

  • Untuk mengatur informasi pribadi Anda agar Git dapat digunakan, jalankan:

    git config --global user.name "John Doe"

    git config --global user.email "john.doe@gmail.com"

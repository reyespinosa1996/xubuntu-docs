Einführung in die Xubuntu Webpräsenz

Xubuntu.org

Die Webseite ist das Haupt-Arbeitsergebnis für die Benutzer. Der Inhalt auf der Webseite soll einfach verständlich und zugänglich sein.

Die Webseite ist eine WordPress-Instanz auf einem Server, welcher von dem Canonical IS gewartet wird. Der Inhalt der Webseite wird vom Xubuntu Website Team beaufsichtigt und verwaltet. Das Website Team kann bei Problemen Tickets an das Canonical IS schicken. Fehler im Bezug auf die Webseite können per Launchpad übermittelt werden, unter Bugs for xubuntu-website.

The source code for the website is available at lp:xubuntu-website (the theme) and lp:xubuntu-website/plugins (the plugins).

Xubuntu-Wiki

The Xubuntu wiki is used for some developer communication and serves as an archive for older release roadmaps and specification.

Packages to be translated

These are packages that the Xubuntu team considers important, and should take precedence when translating.

Packages that only exist in Xubuntu

These packages only exist in Xubuntu, so their translation is solely dealt with by Xubuntu translators. These are of high importance for the Xubuntu team.

  • Xubuntu-Dokumentation

  • Xubuntu-Voreinstellungen

  • Ubiquity-Diashow für Xubuntu

Used by Xubuntu

These packages are used by Xubuntu and provide features to the Xubuntu experience. When these are translated, the base system is immediately more usable for more people.

  • Catfish

  • LightDM GTK+ Greeter

  • LightDM GTK+ Anmeldebildschirm-Einstellungen

  • MenuLibre

  • Mugshot (»Persönliche Informationen«)

  • Whisker-Menü

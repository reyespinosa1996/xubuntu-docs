Présentation de la présence web Xubuntu

Xubuntu.org

Le site web est la référence principale pour l'utilisateur final. Le contenu du site web doit être facilement compréhensible et abordable.

Le site web est une instance WordPress sur un serveur maintenu par Canonical IS. Le contenu du site web est relu et administré par l'équipe du site web Xubuntu. Elle peut soumettre des tickets à Canonical IS en cas de problème. Les bogues relatifs au site web peuvent être soumis sur Launchpad, via le lien des bogues pour xubuntu-website.

The source code for the website is available at lp:xubuntu-website (the theme) and lp:xubuntu-website/plugins (the plugins).

Wiki Xubuntu

The Xubuntu wiki is used for some developer communication and serves as an archive for older release roadmaps and specification.

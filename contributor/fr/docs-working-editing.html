Editing the documentation

First make sure you have the Bzr branch lp:xubuntu-docs and the build dependencies for xubuntu-docs.

Changes to the user documentation should be made to the .xml files under user-docs/C/. Once edited and saved, push changes to your local branch as detailed in Using Bazaar (Bzr).

Similarly, changes to the contributor documentation should be made to the .xml files under contributor-docs/C/.

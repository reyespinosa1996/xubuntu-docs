Paramétrage de Git

Pour paramétrer Git, procédez ainsi :

  • Pour vous assurer que Git est installé dans votre système, exécutez :

    sudo apt-get install git

  • Pour paramétrer les informations personnelles que Git devra utiliser, exéxutez :

    git config --global user.name "John Doe"

    git config --global user.email "john.doe@gmail.com"
